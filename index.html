Mass Storage

Mass storage is long-term storage like disks, CDs, tapes, and flash drives. It holds lots of data permanently but is slower than main memory because it often uses moving parts.

Magnetic Disks (HDD)

Spinning platters, read/write heads, stores lots of data.

Optical (CD/DVD/Blu-ray)

Uses lasers to read a spiral track.

Flash (USB / SD)

No moving parts, fast storage.

Magnetic Tape

High capacity, slow access.
